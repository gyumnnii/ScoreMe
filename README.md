# C/C++ 채점 프로그램

결과값을 비교하여 자동 채점 (Success, Fail) 출력해주는 프로그램

실행 shell 에서 success, fail을 출력하는 형식으로 이루어져 있음  
(이후 CSV 파일로 출력해주는 업데이트 구상 )


## 사용법
```
./static => input.txt / output.txt => 테스트 케이스 및 출력 답
./hw => 소스코드 (.c, .cpp 파일)
./output => 소스코드로 실행된 결과 파일 저장소
```
